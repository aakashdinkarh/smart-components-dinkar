### SegmentedTabs Component

The SegmentedTabs component is a versatile and customizable React component that allows you to create tabbed interfaces with smooth transition effects. This component is ideal for creating interactive user interfaces with organized content sections.

```javascript
import { SegmentedTabs } from 'segmentedTabs';

<SegmentedTabs defaultActiveTab="tab1" onTabChange={onChange}>
  <SegmentedTabs.Tab name="tab1" title="Tab 1">Tab 1</SegmentedTabs.Tab>
  <SegmentedTabs.Tab name="tab2" title="Tab 2">Tab 2</SegmentedTabs.Tab>
  <SegmentedTabs.Tab name="tab3" title="Tab 8">Tab 99</SegmentedTabs.Tab>
</SegmentedTabs>
```
